The SoulFit

This is the proposed system for StateFarm's Problem statement. We estimated the information stored in various attributes of a typical customer in a database such as; Age, BMI, noOfChildren, Profession, Income, savings, etc. into our Learning Model.

Based on the Adjusted Model Parameter Estimation and Statistical Inference, The BMI was discerened to contain the most information and most significant variable in Model Building.

Working

The SoulFit Insurance and Wellness tracker uses GCP for real time data logging and deploying the machine learning model on the cloud. The Python Script combines image processing with a powerful classifier algorithm to accurately predict the person's BMI based on his facial features.

Along with the BMI, other factors are combined using the same information gain used to identify the most significant parameter to arrive at a wellness score, which indicates how fit or unfit a person is. Based on continual tracking of this data in a periodic fashion gives us a trend of this score, which can help identify potential cusotmer behaviour patterns based on well-being and fitness.
